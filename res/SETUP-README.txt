SETUP

Unzip the zip file and find the .jar file in the res folder.
Input “java -jar stocks.jar” in the terminal to run the java file.

INSTRUCTIONS
To get started with the program, you have to create a portfolio. 

1. When you input “1”, you will get two options to create the portfolio - create a flexible portfolio or create a locked portfolio.
2. Any of the option will present you with 2 options to create a portfolio - through a console or a CSV file. 
3. In case of input through the console, You will have to enter a valid name for the new portfolio. 
4. For the CSV file, you will be needed to provide an accurate name of the CSV file.
5. After entering portfolio name, you will have to add stocks and their quantity to the portfolio and also their respective commision fee for each transaction. 
5. Once a portfolio is created, you will be redirected to the main menu where you can choose to perform different operations with the portfolio. 
6. Add stocks to a flexible portfolio (You cannot add stocks to an inflexible portfolio). Once you provide proper input in the menu, you will proceed to buy stocks. To get started, you will have to enter a portfolio name to which you want to add the stock followed by the ticker of stock to be purchased, followed by the quantity of the stock. Your order will not be processed if the portfolio name already exists. You will be required to confirm whether you want to add the stock at this step. If you wish to continue adding, you will go through the same process as mentioned above else the portfolio gets locked. Once your portfolio gets locked, you will be redirected to the main menu.
7. Sell stocks from a flexible portfolio. To get started, you will have to enter a portfolio name to which you want to sell the stock followed by the ticker of stock to be sold, followed by the quantity of the stock. 
8. View Portfolio Amount By Date - This will show you the current portfolio value at any given date.
9. View Portfolio Composition - This will show you the current composition of the portfolio with the respective stocks and their quantity.
10.View Cost Basis of the Portfolio - This will show you the current cost basis of the portfolio with the respective stocks and their quantity.
11.View Performance Graph - This will show you the current performance graph of the portfolio when two dates are provided to it.
12. Quit - This will exit from the Stock software.
